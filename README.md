# Sri Bakes (Chennai)

Sri Bakes is a dedicated home bakery based in Chromepet, Chennai, specializing in bringing your sweetest dreams to life. We believe that a cake should not only look stunning but taste absolutely divine. All our cakes are 100% freshly baked to order using premium ingredients. 

From elegant wedding cakes to cute custom designs for kids, we cater to every special occasion with customizable, bespoke creations.
